Do a full font and design personality upgrade across the entire Skifte Collective Next.js app. The goal: make it feel childlike, joyful, handmade, and warm — like a beloved children's book publisher crossed with a premium mom brand. Think Bobo Choses, Primary.com, and a farmers market zine.

## FONT UPGRADE

### Current: Fredoka + Outfit
### New approach: Layer multiple complementary fonts for personality

1. Install and configure these Google Fonts in src/app/layout.tsx:
   - **Baloo 2** (weight 400, 500, 600, 700, 800) → variable: --font-display → use for BIG hero headlines, section titles, app name
   - **Nunito** (weight 400, 500, 600, 700, 800) → variable: --font-heading → use for card titles, nav items, buttons
   - **Quicksand** (weight 400, 500, 600) → variable: --font-body → use for body text, descriptions, labels

2. Update tailwind.config.ts:
   - display: ["var(--font-display)", "sans-serif"]
   - heading: ["var(--font-heading)", "sans-serif"]
   - body: ["var(--font-body)", "sans-serif"]
   - sans: ["var(--font-body)", "sans-serif"]

3. Update globals.css:
   - --font-heading: 'Nunito', sans-serif
   - --font-body: 'Quicksand', sans-serif
   - Add: --font-display: 'Baloo 2', sans-serif
   - h1 → font-family: var(--font-display)
   - h2, h3 → font-family: var(--font-heading)
   - Remove the letter-spacing: 0.01em rule (let fonts breathe naturally)

4. Update the home page (src/app/page.tsx):
   - Main heading "Swap the things they outgrew." or similar → add className that uses font-display (Baloo 2)
   - Section headings → font-heading (Nunito)
   - Body text → font-body (Quicksand)

5. Update auth page (src/app/auth/page.tsx):
   - The welcome heading → Baloo 2, large, warm
   - Change "Welcome Back" button text to "Welcome back 👋"
   - Change the tagline text in auth to use Baloo 2 for the main line
